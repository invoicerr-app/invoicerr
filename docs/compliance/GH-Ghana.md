## 1. Context & Overview

Ghana has implemented mandatory e-invoicing through the GRA. All businesses must issue electronic invoices for tax compliance.

- **GRA e-Invoice System**
- API integration
- Online portal

### Format Requirements
- **XML format** (GRA schema)
- Digital signature
- Standard format
