## 1. Context & Overview

Zambia has implemented mandatory e-invoicing through the ZRA. All businesses must issue electronic invoices for tax compliance.

## 2. Technical Requirements

### Platform
- **ZRA e-Invoice System**
- API integration
- Online portal

### Format Requirements
- **XML format** (ZRA schema)
- Local standards
- Certification
