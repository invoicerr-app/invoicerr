## 1. Context & Overview

Nigeria has mandatory e-invoicing through the FIRS. All businesses must issue electronic invoices for tax compliance.

## 2. Technical Requirements

- **FIRS e-Invoice System**
- API integration
- TIN verification

### Format Requirements
- **XML format** (FIRS schema)
- PDF format
- Standard invoice format
