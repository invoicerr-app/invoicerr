## 1. Context & Overview

Sri Lanka has implemented mandatory e-invoicing through the IRD. All businesses must issue electronic invoices authorized by the tax authority.

---

## 2. Technical Requirements

### Platform
- **IRD e-Invoice System**
- API integration
- Online portal

### Format Requirements
- **XML format** (IRD schema)
- Digital signature
- Standard format
