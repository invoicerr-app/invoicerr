## 1. Context & Overview

Kenya has mandatory e-invoicing through the KRA iTax system. All businesses must issue electronic invoices for tax compliance.

---

## 2. Technical Requirements

### Platform
- **KRA iTax System**
- API integration
- e-Tax portal

### Format Requirements
- **XML format** (KRA schema)
- PDF format (simplified)
- Standard invoice format

---

## 3. Implementation Timeline

| Date | Scope | Obligation |
| --- | --- | --- |
| **2020+** | Large taxpayers | Progressive mandatory rollout |
| **2024+** | All businesses | Full mandatory e-invoicing |

---

## 4. Business Model

- **Clearance Model:** Pre-registration required
- Integration with VAT system
- Real-time validation

---

## 5. Key Requirements

- PIN (Personal Identification Number)
- KRA certification
- XML/PDF format
- Authorized software
- Real-time submission
