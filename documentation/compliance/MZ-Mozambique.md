# 🇲🇿 Mozambique - E-Invoicing Specifications

**Status:** 🟢 **Mandatory** | Active
**Authority:** AT (Autoridade Tributária)
**Platform:** e-Invoice System

---

## 1. Context & Overview

Mozambique has implemented e-invoicing requirements through the AT. All businesses must issue electronic invoices for tax compliance.

---

## 2. Technical Requirements

### Platform
- **AT e-Invoice System**
- API integration
- National platform

### Format Requirements
- **XML format** (national schema)
- Digital signature
- AT standard
