# E-Invoicing Compliance

Invoicerr maintains a set of reference notes on e-invoicing regulations across 77 countries — covering whether e-invoicing is mandatory, the responsible tax authority, the clearance platform/model, and the relevant technical workflow.

These notes live directly in the repository at [`docs/compliance/`](https://github.com/invoicerr-app/invoicerr/tree/main/docs/compliance) rather than being duplicated into this documentation site, so they stay easy to update via pull request alongside the rest of the codebase.
