# Reçus

La page **Receipts** vous permet d'émettre des documents de preuve de paiement. Les reçus sont créés à partir de [factures](invoices.md) existantes, donc chaque reçu est lié à une facture spécifique.

## Actions

- **Create receipt** — générer un reçu depuis une facture (disponible dans le menu d'actions de la facture)
- **Search** — rechercher par numéro ou nom du client
- **Download** — sauvegarder le reçu au format PDF
- **Send by email** — envoyer le reçu par e-mail en PDF
- **Delete** (icône poubelle) — supprimer un reçu

## Créer un reçu

1. Allez dans la page **Invoices** et trouvez une facture payée ou partiellement payée.
2. Cliquez sur **Create receipt** (icône reçu) sur cette facture.
3. Sélectionnez les lignes et montants à inclure — vous pouvez émettre un reçu pour le montant total ou un paiement partiel.
4. Cliquez sur **Save** pour générer le reçu.

## Statuts

Les reçus n'ont pas de statuts de workflow. Une fois créés, ce sont des documents finaux.

## Reçus partiels

Si un client paie en plusieurs fois, vous pouvez créer plusieurs reçus pour la même facture, chacun couvrant un sous-ensemble de lignes ou un montant partiel.

## Premier usage

Les reçus deviennent disponibles dès que vous avez au moins une facture avec des paiements enregistrés.
