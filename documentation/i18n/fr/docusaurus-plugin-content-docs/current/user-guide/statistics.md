# Statistiques

La page **Statistics** (route `/stats`) vous donne un aperçu visuel de vos revenus dans le temps.

## Ce que vous voyez

- **Graphique des revenus** — tracé sur une plage de dates configurable
- **Indicateurs clés** — revenu total, nombre de factures payées, solde impayé
- **Filtre par période** — basculez entre les vues mensuelle, trimestrielle ou annuelle

## Utilisation

La page est en lecture seule et récupère les données de vos factures payées. Elle se met à jour automatiquement quand vous marquez des factures comme **Paid** ou en créez de nouvelles.

## Premier usage

Le graphique sera vide jusqu'à ce que vous ayez au moins une facture payée. Une fois que vous [marquez une facture comme payée](invoices.md#statuts), la page des statistiques commence à se remplir.
