# Entreprise & identité visuelle

## Company

Votre identité commerciale — ces informations apparaissent sur chaque document émis.

- **Name** (obligatoire)
- **VAT Number** (optionnel)
- **Legal ID / SIRET** (optionnel)
- **Adresse** — Rue, Complément d'adresse, Code postal, Ville, État / Province, Pays
- **Email** et **Phone**
- **Website**
- **Currency** — devise par défaut pour les nouveaux documents
- **Date Format**
- **Logo** — téléchargez le logo de votre entreprise (apparaît sur les PDF)
- **Exempt VAT** — bascule pour désactiver la TVA sur tous les documents

### Numérotation des documents

- **Quote Number Format** et **Starting Number**
- **Invoice Number Format** et **Starting Number**
- **Receipt Number Format** et **Starting Number**

### Format PDF des factures

- Choisissez le format de facturation électronique par défaut : **PDF**, **Factur-X**, **ZUGFeRD**, **XRechnung**, **UBL** ou **CII**

## PDF Templates

Personnalisez l'apparence de vos documents PDF, avec un aperçu en direct que vous pouvez basculer entre les mises en page Facture, Devis et Paiement :

- **Typography** — police de caractères utilisée dans tout le PDF
- **Logo et couleur** — téléchargez le logo de votre entreprise et choisissez la couleur d'accent utilisée pour l'en-tête du tableau et les mises en valeur
- **Espacement & mise en page** — marges et padding de la page
- **Labels** — personnalisez le texte affiché pour chaque champ (ex. *Bill to*, *VAT*, *Total*), utile pour traduire les documents dans une autre langue
