# Dashboard

Le **Dashboard** est la première page que vous voyez après vous être connecté. Il vous donne un aperçu de votre activité en un coup d'œil.

## Ce que vous voyez

- Message de bienvenue **Welcome back** avec votre nom
- **Revenus** — gains du mois et de l'année en cours
- **Cartes de résumé** — total des devis, factures et clients avec leurs décomptes
- **Activité récente** — les 5 devis et factures les plus récents

## Utilisation

Chaque carte de résumé et élément récent est cliquable — cliquez sur un devis pour l'ouvrir, ou sur un décompte pour accéder à la liste complète.

## Pas encore de données

Si vous débutez, le dashboard affiche des messages d'état vide vous guidant vers vos premières actions. Consultez la [Vue d'ensemble](overview.md) pour l'ordre de démarrage recommandé.
