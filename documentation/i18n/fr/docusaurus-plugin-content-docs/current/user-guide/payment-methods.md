# Moyens de paiement

La page **Payment Methods** stocke les moyens réutilisables par lesquels vos clients peuvent vous payer — par exemple un compte bancaire professionnel ou une adresse PayPal. Vous les définissez une fois, puis les associez aux devis et factures.

## Actions

- **Add payment method** — créer un nouveau moyen de paiement
- **Search** — rechercher par nom, détail ou type
- **Filter** — basculer les badges **Active** / **Inactive**
- **View** (icône œil) — détails en lecture seule
- **Edit** (icône crayon) — modifier un moyen
- **Delete** (icône poubelle) — supprimer un moyen (demande confirmation)

## Créer un moyen de paiement

Cliquez sur **Add payment method** et remplissez :

- **Name** (obligatoire) — un libellé que vous reconnaîtrez, ex. *"Virement bancaire (pro)"*
- **Details** (optionnel) — texte libre comme un IBAN ou numéro de compte
- **Type** (obligatoire) — **Bank transfer**, **PayPal**, **Check**, **Cash** ou **Other** (Bank transfer par défaut)

## Statuts

| Statut | Signification |
| --- | --- |
| **Active** | Sélectionnable sur les nouveaux devis et factures |
| **Inactive** | Masqué des nouveaux documents, conservé pour l'historique |

## Premier usage

Sans aucun moyen de paiement, la page affiche *"No payment methods yet."* et un bouton **Add payment method**. Une fois que vous avez au moins un client et un moyen de paiement, vous êtes prêt à créer un [devis](quotes.md).
